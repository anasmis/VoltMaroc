Setup Instructions

  1. Make sure your Strapi backend is running
  2. Update js/strapi-config.js with your Strapi URL and API token
  3. Configure CORS in Strapi to allow requests from this domain
  4. Enable public read permissions for Articles, Categories, and Tags
  5. Refresh this page and test the connection

Quick Configuration:
Edit js/strapi-config.js and update:
API_URL: 'http://localhost:1337'
API_TOKEN: 'your-token-here'
